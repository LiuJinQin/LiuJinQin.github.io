    • 【读书笔记】《神经网络与深度学习》邱锡鹏

    ch01 绪论

    1.1 人工智能

    人工智能（Artificial Intelligence，AI）：让机器拥有人的智能（感知、学习、认知）。

    • 感知：（计算机视觉、语音信息处理）
    • 学习：（监督学习、无监督学习、强化学习）
    • 认知：（知识表示、自然语言理解、推理、规划、决策）

    ![图 1.1 人工智能发展史](图 1.1 人工智能发展史.png)

    1.2 机器学习

    机器学习（Machine Learning，ML）：从数据中学习出一般规律，利用规律进行新的预测。

    实际任务中使用机器学习模型的几个步骤：

    1. 数据预处理：去掉一些缺失特征的样本、去掉一些冗余的数据特征、对数值特征进行缩放和归一化
    2. 特征提取：提取高质量特征。如文本分类中去除停用词
    3. 特征转换：对特征降维（特征抽取和特征选择）和升维。常用方法：主成分分析（Principal Components Analysis，PCA）、线性判别分析（Linear Discriminant Analysis，LDA）等
    4. 预测：学习一个函数并进行预测

    ![图 1.2 传统机器学习的数据处理流程](图 1.2 传统机器学习的数据处理流程.png)

    很多机器学习问题往往变成了特征工程（Feature Engineering）问题

    1.3 表示学习

    机器学习中表示特征的两种方式：

    • 局部表示 / 离散表示 / 符号表示：one-hot向量形式
    • 分布式表示：低维稠密向量 。

    嵌入（Embedding）：我们使用神经网络将高维的局部表示空间 $\R^{|D|}$ 映射到一个非常低维的分布式表示空间$R^D$，$D\ll|V|$。机器学习中此过程称为嵌入。自然语言中词的分布式表示，经常叫做词嵌入。

    ![图 1.3 one-hot向量空间与嵌入空间](图 1.3 one-hot向量空间与嵌入空间.png)

    表示学习（Representation Learning）：可以自动学习有效特征，提高机器学习模型性能的学习方法。

    表示学习的关键：构建一定深度的多层次特征表示。因为要学习到一个好的高层语义表示（一般为分布式表示），通常要从底层特征开始经过多次非线性转换才能得到，因此选择深层结构。深层结构可以增加特征的重用性，从而指数级地增加表示能力。

    传统机器学习的特征学习方法：主成分分析、线性判别分析、独立成分分析。

    1.4 深度学习

    深度学习（Deep Learning，DL）：从数据中学出一个深度模型。从数据中自动学习有效的特征表示。

    深度学习早期主要用来进行表示学习，后来越多越多地用于处理更复杂的推理、决策问题。

    ![图 1.4 深度学习的数据处理流程](图 1.4 深度学习的数据处理流程.png)

    深度学习所采用的的模型是神经网络模型（当然还有其他的模型），主要原因是神经网络模型可以使用误差反向传播算法，可以比较好地解决贡献度分配问题（Credit Assignment Problem，CAP）——一个系统中不同的组件或其参数对最终系统输出结果的贡献或影响。

    1.5 神经网络

    神经网络，是人工神经网络的简称，即一种模仿人脑神经网络的数学模型，由节点（人工神经元）互相连接组成。不同节点间的连接被赋予不同的权重，每个权重代表一个节点对另一个节点的影响大小。每个节点都代表一种特定的函数，由其他节点传来的信息经过相应权重进行计算，再输入到一个激活函数中得到一个新的节点值。

    1.7 常用的深度学习框架

    为什么需要框架：深度学习中，一般用误差反向传播算法进行参数学习，手工求梯度再写代码的方式非常低效。此外，深度学习模型的计算资源很多，需要在CPU和GPU之间无缝切换。因此，我们需要一种可以自动求梯度，且可以无缝在CPU和GPU之间切换的功能，于是产生了支持这些功能的深度学习框架。代表性框架：Theano、Caffe、TensorFlow、 PyTorch、飞桨（PaddlePaddle）、Chainer和MXNet等。

    • Theano：蒙特利尔大学。停止维护
    • Caffe：加州伯克利分校。针对卷积神经网络的计算框架，主要用于计算机视觉。（Caffe2已并入PyTorch）
    • TensorFlow：Google 。
    • PyTorch：Facebook、NVIDIA、Twitter。
    • 飞桨（PaddlePaddle）：百度。
    • MindSpore：华为。用于端边云场景
    • Chainer：日本Preferred Networks。非常适合进行一些复杂的决策或推理任务。
    • MXNet：亚马逊、华盛顿大学和卡内基·梅隆大学。MXNet支持混合使用符号和命令式编程。

    模块化神经网络框架：建立在基础框架之上的高度模块化的神经网络库。

    • 基于TensorFlow和Theano的Keras（Keras已经被集成到TensorFlow 2.0版本中）
    • 基于Theano的Lasagne
    • 面向图结构数据的DGL

    1.8 总结与深入阅读

    传统机器学习中，开发一个实际的机器学习系统，除了模型和算法，人们往往花费大量的精力去做特征工程。如何自动学习有效的数据表示成为了机器学习中的关键问题。早期的表示学习方法，如特征抽取和特征选择，都是人工引入特征进行学习，这种表示学习并不是端到端的学习，且效果不能保证有效。为了解决这一问题，提出了深度学习。

    深度学习里要解决的问题是贡献度分配问题，为了解决这一问题，发现神经网络模型恰好有效。

    目前，深度学习主要以神经网络模型为基础，研究如何设计模型结构，如何有效地学习模型的参数，如何优化模型性能以及在不同任务上的应用等。

    和深度学习相关的学术会议

    • 国际表示学习会议（International Conference on Learning Representations，ICLR）：主要聚焦于深度学习。
    • 神经信息处理系统年会（Annual Conference on Neural Information Processing Systems，NeurIPS）：交叉学科会议，但偏重于机器学习。主要包括神经信息处理、统计方法、学习理论以及应用等。
    • 国际机器学习会议（International Conference on Machine Learning，ICML）：机器学习顶级会议，深度学习作为近年来的热点，也占据了ICML的很大比例。
    • 国际人工智能联合会议（International Joint Conference on Artificial Intelligence，IJCAI）：人工智能领域最顶尖的综合性会议．历史悠久，从1969年开始举办。
    • 美国人工智能协会年会（AAAIConferenceonArtificialIntelligence，AAAI）：人工智能领域的顶级会议，每年二月份左右召开，地点一般在北美。

3.3 数学公式

Q：Hexo默认使用"hexo-renderer-marked"引擎渲染网页，该引擎会把一些特殊的markdown符号转换为相应的html标签

A：没有用作者的方法，因为按作者的方法，我会报错：Cannot find module ‘markdown-it-katex’，但明明hexo-renderer-markdown-it-plus插件就表示katex渲染器，而且安装的插件是想使用katex渲染器，最后却用mathjax: true，表示使用的是mathjax渲染器，我自己感觉作者这里是有点混乱的。于是我找了另一个方法：https://wty-yy.github.io/posts/3849/index.html，步骤如下：

  1. 卸载原有的渲染器：npm un hexo-renderer-marked --save。

  2. 安装katex渲染器：npm i hexo-renderer-markdown-it-plus --save。

    注意网上很多教程都使用mathjax渲染器（npm install hexo-renderer-kramed --save），但mathjax也有一些问题，如把\ \转义成\，所以最终使用katex渲染器。此处npm list检查有没有安装hexo-renderer-kramed，如果有，也卸载（npm un hexo-renderer-kramed --save）

  3. 安装hexo-math插件：npm install hexo-math --save。作者说是为了让字体变好看？应该不止，后面配置中也用到了，安装吧。

  4. 在“站点配置文件”末尾添加代码

  5. “主题配置文件”末尾添加代码

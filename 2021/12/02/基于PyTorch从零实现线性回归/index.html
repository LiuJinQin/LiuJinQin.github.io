深度模型的四大基本要素

1.模型（model）

模型就是y关于x的表达式，线性回归中y与x之间是线性关系，表达式例： y^=x1w1+x2w2+b\hat y=x_1w_1+x_2w_2+by^​=x1​w1​+x2​w2​+b。在代码中定义函数linreg()表达这个模型，输入特征X、权重w，偏差b，输出预测 y^\hat yy^​。

注意，我们知道X一般是矩阵，w一般是向量，那为什么不用torch.mv函数做矩阵向量积呢？其实可以用。这里用的是torch.mm函数，这个函数根据传入参数的张量维度有很多重载函数，既可以做点积相当于torch.dot函数，也可以做矩阵向量积相当于torch.mv函数，更可以做矩阵乘法，所以这里我们使用这个功能更全面的torch.mm函数。

2.训练数据（training set）

Q：我们需要构建什么样的数据集？需要设置哪些变量？

A：根据上述模型的形式，我们最起码要定义特征X、权重w、偏差b、标签y这些变量。在代码中，可以这样定义：

样本数：n_samples

特征数：n_features

特征：features，形状（n_samples，n_features）

权重：w，形状（n_features，1）

偏差：b，标量

标签：labels，形状（n_samples，1）

3.损失函数（loss function） — 平方损失（square loss）

衡量误差的函数称为损失函数。这里使用的平方误差函数也被称为平方损失（square loss）。通常，我们用训练数据集中所有样本误差的平均来衡量模型预测的质量，即：

l(w1,w2,b)=1n∑i=1nl(i)(w1,w2,b)=1n∑i=1n12(y^(i)−y(i))2\begin{aligned} l(w_{1},w_{2},b) &= \frac{1}{n}\sum_{i=1}^{n}l^{(i)}(w_{1},w_{2},b)\\\\ &= \frac{1}{n}\sum_{i=1}^{n}\frac{1}{2}(\hat{y}^{(i)} - y^{(i)})^{2} \end{aligned} l(w1​,w2​,b)​=n1​i=1∑n​l(i)(w1​,w2​,b)=n1​i=1∑n​21​(y^​(i)−y(i))2​

在代码中我们定义函数squared_loss()去表示损失函数，输入预测y^\hat yy^​、标签yyy，输出损失值losslossloss。

4.优化算法 — 小批量随机梯度下降（mini-batch stochastic gradient descent）

小批量随机梯度下降算法：我们先选取一组模型参数的初始值，例如随机选取；接下来对参数进行多次迭代，使得每次迭代都可能降低损失函数的值。在每次迭代中，我们先随机均匀采样一个由固定数目训练数据样本所组成的小批量（mini-batch）BBB；然后求小批量中数据样本的平均损失有关模型参数w、b的导数（梯度）；最后用此结果与预先设定的一个正数的乘积作为模型参数在本次迭代的减小量。

在训练本节讨论的线性回归模型的过程中，模型的每个参数将作如下迭代：

w1←w1−η∣B∣∑i∈B∂l(i)(w1,w2,b)∂w1=w1−η∣B∣∑i∈Bx1(i)(x1(i)w1+x2(i)w2+b−y(i))w_{1}\leftarrow w_{1} - \frac{\eta }{\left | B \right |}\sum_{i\in B}^{ } \frac{\partial l^{(i)}(w_{1},w_{2},b)}{\partial w_{1}} = w_{1} - \frac{\eta }{\left | B \right |}\sum_{i\in B}^{ } x_{1}^{(i)}(x_{1}^{(i)}w_{1} +x_{2}^{(i)}w_{2} + b - y^{(i)}) w1​←w1​−∣B∣η​i∈B∑​∂w1​∂l(i)(w1​,w2​,b)​=w1​−∣B∣η​i∈B∑​x1(i)​(x1(i)​w1​+x2(i)​w2​+b−y(i))

w2←w2−η∣B∣∑i∈B∂l(i)(w1,w2,b)∂w2=w2−η∣B∣∑i∈Bx2(i)(x1(i)w1+x2(i)w2+b−y(i))w_{2}\leftarrow w_{2} - \frac{\eta }{\left | B \right |}\sum_{i\in B}^{ } \frac{\partial l^{(i)}(w_{1},w_{2},b)}{\partial w_{2}} = w_{2} - \frac{\eta }{\left | B \right |}\sum_{i\in B}^{ } x_{2}^{(i)}(x_{1}^{(i)}w_{1} +x_{2}^{(i)}w_{2} + b - y^{(i)}) w2​←w2​−∣B∣η​i∈B∑​∂w2​∂l(i)(w1​,w2​,b)​=w2​−∣B∣η​i∈B∑​x2(i)​(x1(i)​w1​+x2(i)​w2​+b−y(i))

b←b−η∣B∣∑i∈B∂l(i)(w1,w2,b)∂b=b−η∣B∣∑i∈B(x1(i)w1+x2(i)w2+b−y(i))b\leftarrow b - \frac{\eta }{\left | B \right |}\sum_{i\in B}^{ } \frac{\partial l^{(i)}(w_{1},w_{2},b)}{\partial b} = b - \frac{\eta }{\left | B \right |}\sum_{i\in B}^{ } (x_{1}^{(i)}w_{1} +x_{2}^{(i)}w_{2} + b - y^{(i)}) b←b−∣B∣η​i∈B∑​∂b∂l(i)(w1​,w2​,b)​=b−∣B∣η​i∈B∑​(x1(i)​w1​+x2(i)​w2​+b−y(i))

其中超参数（hyperparameter）：

∣B∣\left | B \right |∣B∣：批量大小（batch size)，每个小批量中的样本个数。

η\etaη：学习率（learning rate)，取正数。

我们通常所说的“调参”指的正是调节超参数，例如通过反复试错来找到合适的超参数。

代码中，定义函数sgd()表示优化算法，输入参数params（形式是[w,b]）、学习率lr、批量大小batch_size，根据上述公式形式，输出优化后的参数。

tensor数据操作

在PyTorch中，张量tensor是存储和变换数据的主要工具。tensor提供GPU计算和自动求梯度等更多功能，这些使得tensor更加适合深度学习。具体如何使用参阅《PyTorch 中文手册》和《动手学深度学习》第2章预备知识。

autograd包自动求梯度

参阅自《动手学深度学习》2.5自动求导一节

一个例子：对函数y=2xTxy=2x^Txy=2xTx关于列向量x求导

此时我们想计算x的另一个函数对x的梯度

当y不是标量时，向量y关于向量x的导数的最自然解释是一个矩阵。对于高阶和高维的y和x，求导的结果可以是一个高阶张量。但我们的目的不是计算出微分矩阵，而是批量中每个样本单独计算的偏导数之和。

1 生成数据集

前面讲基本要素的第2项里说明了要定义的变量，在此再陈列一下：

样本数：n_samples。=1000

特征数：n_features。=2

特征：features，形状（n_samples，n_features）。（1000,2）

权重：w，形状（n_features，1）。（2,1）

偏差：b，标量。（1,）

标签：labels，形状（n_samples，1）。（1000,1）

这里我们把标签的形式改动一下，增加一个随机噪声项 ε\varepsilonε。

随机噪声：ε\varepsilonε ，形状（n_samples，1）。服从均值为0、标准差为0.01的正态分布。

最终标签labels形式为： y=Xw+b+εy=Xw+b+\varepsilony=Xw+b+ε。

另外，这里的参数w，b是自己设置的用来生成训练集的参数。我们后续还要初始化一个用于训练模型的参数w，b，这个初始化的参数才是我们后续要训练的参数，训练后的参数会逼近我们现在设置的这两个参数，所以我们称现在设置的这个是真实参数，相当于我们提前知道了优化后的参数大概是什么数值，这也是因为我们在人工设计生成数据集。如果数据集不是我们人工生成的，那其中待训练的参数会训练成什么样我们是不知道的。注意区分这其中的区别。

2 读取数据集

我们不是一次训练完所有的数据集，而是分成很多小批量数据样本去训练，因此我们需要一个函数，把训练集样本划分成批量大小为batch_size的若干样本。函数名定义为data_iter()，输入想要设置的批量大小batch_size、特征features、标签labels，输出batch_size个随机样本的特征和标签。

3 初始化模型参数w、b

这里的w、b才是我们要经过模型训练的参数，如果训练的好，会无限逼近我们先前设置的用于生成训练集的真实参数true_w，true_b。

7 训练模型

在训练之前，我们应先定义两个超参数：学习率（lr） 和批量大小 （batch_size），提示下这里的批量大小在前面定义了。在训练中，我们将多次迭代模型参数，以达到更好的优化效果，所以还需要定义一个迭代周期个数变量（n_epochs）。

在每一个迭代周期（epoch）中，我们根据当前读取的小批量数据样本（特征X和标签y），记录每个小批量的损失lll，通过调用反向函数backward计算小批量随机梯度，并调用优化算法sgd迭代模型参数。在这一个迭代周期中，我们完整地遍历了一遍data_iter函数，对训练数据集中所有样本都使用一次（假设样本数能够被批量大小整除)，使用完所有样本后，我们用优化后的参数代入模型计算出预测值，然后与标签值比较，查看损失情况。

训练完成后，我们可以比较学到的参数和用来生成训练集的真实参数，它们应该很接近。
